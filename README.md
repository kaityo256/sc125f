# sc125f

## 概要

川合さんによる数独難易度判定プログラム。

`const char*`で数独を渡すと、難易度をintで返す。

```cpp
#include "sc125fa.hpp"
int main(void) {
  const char *str = "000000000000000001001023040000500020002041600070000000004036702060050030800900060";
  sc125f::initTables(); // 定数テーブルの初期化.
  int sc = sc125f::calc(str);
  printf("%d\n", sc); // => 16826
}
```

## ライセンス

すべて川合さんが保持しており、公開しない。
